## Options

To change options you need to set ```data-eu-config='{"url": "#"}'```


### Defaults
```josn
{
  cookieKey: 'eu_cookie',
  showAfter: 365,
  showDeclinedAfter: 6 * 30, // 6 months
  analyticsValue: 'analytics',
  marketingValue: 'marketing',
  url: './pravidla-cookies',
},
```
